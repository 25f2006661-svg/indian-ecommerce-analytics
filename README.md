\# Indian E-Commerce Analytics Dashboard



\## Project Overview

This project analyzes Indian e-commerce sales data using Power BI, Python, SQL, and Exploratory Data Analysis techniques.



The dashboard provides insights into:

\- Revenue trends

\- Category performance

\- Customer behavior

\- Regional sales analysis

\- Monthly revenue growth



\---



\## Tools \& Technologies

\- Power BI

\- Python

\- Pandas

\- Matplotlib

\- SQL

\- Jupyter Notebook



\---



\## Project Workflow

1\. Data Cleaning

2\. Exploratory Data Analysis (EDA)

3\. Dashboard Development

4\. Business Insights Generation



\---



\## Dashboard Features

\- KPI Cards

\- Monthly Revenue Trend

\- Revenue by Category

\- Revenue by Zone

\- Gender-based Sales Analysis

\- Interactive Filters \& Slicers
